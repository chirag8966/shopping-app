Shopping App Documentation

Overview

This shopping application provides a modern e-commerce experience with a responsive design, interactive shopping cart, special offers integration, and easy navigation. The application is built using a modern tech stack with best practices in mind.

Front-End Technologies

Angular 19

Latest version with improved performance and features like Angular Signals for state management.

Tailwind CSS

Utility-first CSS framework for responsive design and rapid UI development.

PrimeNG

Rich set of UI components with built-in accessibility and theming options.

RxJS

Reactive programming library for handling asynchronous operations and events.

Advanced Concepts

  • Angular Signals: Reactive state management with improved performance over traditional observables.
  • RxJS Operators: Utilizing operators like forkJoin, map, and switchMap for efficient API calls.
  • Lazy Loading: Modules loaded on demand to improve initial load time.
  • Responsive Design: Mobile-first approach ensuring the app looks great on all devices.
  • Theming Support: Dynamic theme switching with PrimeNG theme integration.

Available APIs

Endpoint Method Description
/products GET Fetch all products from the database
/products/:id GET Fetch a specific product by ID
/offers GET Fetch all available special offers
/offers/:id GET Fetch a specific offer by ID

Getting Started

Frontend

cd front-end && npm install && npm start

Backend

cd back-end && npm install && npm run seed && npm run start:dev

The frontend will be available at http://localhost:4200 and the backend API at http://localhost:3000.
